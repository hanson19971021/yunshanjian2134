環境介紹

金針山景觀

金針山位於台東太麻里，海拔約800公尺，是台灣著名的金針花產地。每年8月至9月間，整片山頭被金黃色的金針花海覆蓋，形成壯觀的自然景觀。

除了金針花季外，這裡四季都有不同的自然風光。春季有櫻花綻放，夏季綠意盎然，秋季有楓紅點綴，冬季則有雲海繚繞。

從民宿的觀景台可以欣賞到太平洋的壯麗景色，日出時分，陽光灑在海面上，形成一幅令人屏息的美景。

訂房注意事項

  • • 入住時間：下午3:00後，退房時間：上午11:00前。

  • • 為響應環保並配合政府政策，民宿不提供一次性使用盥洗用具，請自備個人盥洗用品或可向櫃檯購買。
  • • 入住房客均享有免費早餐,每間房間均有免費Wi-Fi。
  • • 請愛惜民宿內相關設施，如有任何損毀或擅自帶走任何物品，民宿將依規定向房客照價索賠。
  • • 為維護住宿品質，除規定區域外，民宿全面禁止吸菸，違者將依法檢舉，予以處分。
  • • 為維護住宿品質，請依房型人數進住，如需加人請事先告知，加床為一人NT$800元。
  • • 為維護住宿品質，客房內請勿大聲喧嘩，並於晚上10點後請降低音量。
  • • 未經允許嚴禁於民宿內使用火源，並禁止於房間內進行煮食之行為。
  • • 民宿內嚴禁任何非法行為（如打架、賭博等）、或攜帶任何違禁品（毒藥、危險物品等），如發現即報警處理。
  • • 退房時，除確實將個人用品帶走，並應繳回房卡及鑰匙，如有遺失應支付NT$300元。
  • • 本場所如有張貼『禁止進入』標語，請各位住客勿隨意進入。

寵物友善

  • • 如有攜帶寵物入住者，入住當日每房將收取保證金NT$1000元，退房時，房間若有髒汙(床被單有寵物痕跡或隨意大小便)或設備造成破壞，將由保證金扣除相應的金額，若破壞程度超過保證金金額，將會額外申請賠償，若無任何破壞或損失，將全額退還保證金。
  • • 每房最多2隻寵物入住，需於訂房告知，若有與入住當日數量不同者，本民宿有權拒絕入住。
  • • 為維護住宿品質，每隻寵物將收取NT$300元清潔費(本民宿不提供寵物用品)
  • • 請確認您的寵物已施打基本疫苗，且無跳蚤、傳染病或任意攻擊性行為。
  • • 請勿讓寵物上床或使用民宿的枕頭、棉被（請自行攜帶寵物墊/床）。

取消政策

  • • 入住前7天取消訂房，訂金扣除手續費NT$100元後之餘額將全額退款

  • • 入住前1-2天取消訂房，退還已付訂金20%
  • • 入住當天取消或未入住者，將不退還訂金
  • • 取消或更改訂房需於一周前通知
  • • 如遇天災等不可抗力因素(以當地縣市政府宣布為準)，可以選擇退還訂金(訂金扣除手續費NT$100元後餘額將全額退款)或保留3個月。房客須於住宿前一日中午12點提出，逾時將不予受理。

關於我們

我們的故事

雲山間民宿始於2024年，源自於一個簡單的想法 —— 在這片寧靜的土地上，為旅人打造一個溫暖的家。

我們精心挑選了這片寧靜的地段，遠離城市的喧囂，讓每位住客都能享受到大自然的寧靜與美好。在這裡，您可以放慢腳步，享受生活的點滴。

我們相信，旅行不僅是觀光，更是一種生活體驗。因此，我們致力於為每位旅客提供最舒適的住宿環境和服務。

舒適體驗

我們注重每個細節，從床具選擇到空間布置，都以住客的舒適為優先考量。

環保永續

為響應環保，我們不提供一次性用品，為地球盡一份心力。

在地連結

與在地小農合作，為住客提供當季食材與在地體驗。
